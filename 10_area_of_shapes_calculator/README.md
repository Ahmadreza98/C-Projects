# Area of Shapes Calculator in C

## Overview

This project is a simple C program that calculates the **area of basic shapes** based on user input. The program currently supports:

- Circle
- Square

It demonstrates the use of conditional statements, switch-case logic, arithmetic calculations, and input validation in C.

---

## Features

- Calculates the area of a circle or a square
- Uses `switch-case` for shape selection
- Handles invalid or negative input gracefully
- Displays results with two decimal places
- Provides clear prompts and messages

---

## Requirements

- C compiler (e.g., `gcc`)
- Standard C library
- Compatible with Linux, macOS, and Windows

---

## Build Instructions

Compile the program using `gcc`:

gcc area_shapes.c -o area_shapes
