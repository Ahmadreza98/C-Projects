# Sample Calculator in C

## Overview

This project is a simple command-line calculator written in C. It performs basic arithmetic operations—addition, subtraction, multiplication, and division—on two floating-point numbers entered by the user.

The program demonstrates:
- Console input/output
- Function usage
- `switch` statements
- Basic error handling

It is intended as a learning project for beginners in C programming.

---

## Features

- Supports four arithmetic operations:
  - Addition (`+`)
  - Subtraction (`-`)
  - Multiplication (`*`)
  - Division (`/`)
- Uses floating-point numbers for more precise calculations
- Handles division by zero gracefully
- Clear and simple user interaction via the terminal

---

## Requirements

- C compiler (e.g., `gcc`)
- Standard C library
- Compatible with Linux, macOS, and Windows (via MinGW or similar)

---
