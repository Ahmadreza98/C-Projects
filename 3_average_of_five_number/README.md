# Average of Five Numbers in C

## Overview

This project is a simple C program that calculates the average of **five floating-point numbers** entered by the user via the command line. It demonstrates the use of loops, floating-point arithmetic, and basic input validation.

---

## Features

- Accepts five numeric inputs from the user
- Calculates the arithmetic mean
- Uses a `for` loop to avoid code repetition
- Performs basic validation to detect invalid input
- Outputs the result formatted to two decimal places

---

## Requirements

- C compiler (e.g., `gcc`)
- Standard C library
- Compatible with Linux, macOS, and Windows

---

## Build Instructions

Compile the program using `gcc`:

```bash
gcc average.c -o average
