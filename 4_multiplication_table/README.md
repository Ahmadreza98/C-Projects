# Multiplication Table Generator in C

## Overview

This project is a simple C program that generates a **multiplication table** for a number entered by the user. It demonstrates the use of loops, input validation, and basic arithmetic in C. 

---

## Features

- Accepts an integer input from the user
- Prints the multiplication table from 1 to 10
- Handles invalid input gracefully
- Simple and readable console output

---

## Requirements

- C compiler (e.g., `gcc`)
- Standard C library
- Compatible with Linux, macOS, and Windows

---

## Build Instructions

Compile the program using `gcc`:

```bash
gcc multiplication_table.c -o multiplication_table
